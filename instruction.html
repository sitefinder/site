System Instruction

1. Site Search

Use the Site Search feature to find any domain information. Enter the Site ID in the search box and click Search. You can copy, share, or view the location on Google Maps if available.

2. Calculator

The built-in Calculator can perform basic and advanced mathematical operations. Use it to calculate quickly without leaving the app.

3. Developer Info

Access the Developer page to see app version, creator info, and GitHub link.

4. Theme & Customization

Click the palette icon at the top-right corner to select your preferred theme. Themes change the color glow of the Dashboard cards.

5. App Update

If an app update is available, a notification button will appear. Click it to download the latest version from the official link.

6. Logout

Click the TERMINATE SESSION button to log out. Confirm the popup to safely exit.
